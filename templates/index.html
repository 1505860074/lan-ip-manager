局域网设备 IP 管理面板

这个面板是做什么的？ 当你用一根网线，把这台电脑和另一台 Linux 设备（比如机器人、工控机）直接对连， 但对方的 IP 你不知道、或它和你不在同一网段、平时根本连不上时—— 用它就能把对方找出来、确认能登录、再把它的 IP 改成你想要的。

【警告】当前不是以 root 运行，扫描和调整网段会失败，请用 sudo 启动本程序。

1 找到对方设备

先在下面「本机网口」表里，找到「网线」列显示「已连」的那一行—— 那就是插了网线、正连着对方设备的网口。然后：

  1. 点这一行右边的「扫描找设备」按钮。
  2. 程序会在这个口上持续监听对方发出的广播包。
  3. 这个口会自动填进第 ② 步的「本机直连网口」，之后连对方时要用。
  4. 找到对方后，点「停止扫描」结束监听即可。

这张表是「当前在线」的实时视图：

找到你要改的那台后，点它这一行的「选为目标」——它的 IP 会自动填进第 ② 步的「对方当前 IP」；然后点「停止扫描」即可。

几点说明：

    
    
      
  • 改 IP 目前内置的是 netplan 一套流程（Ubuntu 18.04+ 默认的网络配置方式）。

  • 执行前程序会自动备份对方原配置到 /tmp/netplan-bak/；点「显示备份文件」能随时只读地查看这份原始配置（同时完整写入本机日志），万一改错需要手工恢复时，照着它填回去即可。
