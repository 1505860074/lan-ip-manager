局域网设备 IP 管理面板

给「网线直连、但和你不在同一网段」的 Linux 设备改 IP。 按 ① 选网口 → ② 发现设备 → ③ 连接对方 → ④ 修改 IP 四步走，每步都有说明和反馈。

【警告】当前不是以 root 运行，扫描和调整网段会失败，请用 sudo 启动本程序。

1 找到对方设备

在「网线」列显示 已连 的那行——那就是插了网线、连着对方的口——点它那行的 「扫描找设备」。 程序会在这个口上持续监听对方发出的广播包，每秒刷新下方结果，即便网段不同也能发现它、拿到它当前的 IP； 找到对方后点「停止扫描」即可。同时这个口会自动填到右侧 ② 的「本机直连网口」（不同网段时程序靠它自动临时打通）。未插网线的口按钮是灰的、不能扫。

2 测试网络连接

填对方账号密码，点「测试连接」确认能登录。如果本机和对方不在同一网段，程序会在连接时自动临时打通网络、用完自动清理，你不用管——所以不再需要手动「打通链路」这一步。

3 修改对方 IP

    

      填好新 IP，先点“预览”看清将执行的命令，确认无误后再“执行”。执行是全自动的：
      必要时临时打通网络 → 备份 → 写入 → 应用 → 清理临时 IP → 自动 ping 验证新 IP 是否生效，
      全程无需你手动敲命令，最后直接告诉你新 IP 通不通。
